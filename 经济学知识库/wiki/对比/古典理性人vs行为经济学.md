# 古典理性人 vs 行为经济学

> [!note] 一句话
> 古典经济学假设人是冷静的"计算器"，行为经济学发现人是会犯系统性错误的"真实人"。两者**不是对立，而是互补与修正**：行为经济学站在古典模型的肩膀上，把"偏差"补回到地图里。

## 一、两种人性假设的对照

| 维度 | 古典理性人（Homo Economicus） | 真实人（行为经济学） |
| --- | --- | --- |
| 核心假设 | 偏好稳定、追求效用最大化 | 偏好随情境变化，常用直觉走捷径 |
| 认知方式 | 全知全能、无限算力 | 受 [[系统1与系统2]] 切换支配，常靠快思考 |
| 对待得失 | 收益与损失对称看待 | [[损失厌恶]]：失去之痛约为得到之乐的两倍 |
| 价值判断 | 看绝对的最终财富状态 | [[前景理论]]：看相对参照点的变化 |
| 表述无关性 | 同一事实换种说法结论不变 | [[框架效应]]：说法变了，选择就变了 |
| 估值来源 | 由真实价值独立判断 | [[锚定效应]]：被无关数字牵着走 |
| 金钱可替换 | 钱就是钱，可自由调配 | [[心理账户]]：钱被分进不同"心理抽屉" |
| 拥有的影响 | 拥有与否不改变估值 | [[禀赋效应]]：一旦拥有就更舍不得 |
| 是否会拖延 | 永远立即执行最优计划 | [[拖延与自我控制]]：明知该做却拖 |
| 自信程度 | 概率判断校准准确 | [[过度自信]]：高估自己的判断力 |

## 二、它们其实共享同一套底层工具

行为经济学并没有推翻经济学的核心引擎，下面这些概念两边都在用：

- [[激励]]：人会对激励作出反应——只是真实人的反应方式更复杂。
- [[边际分析]]：边际上做决策的逻辑依然成立。
- [[机会成本]]：选择的代价仍是放弃的最高价值选项；但真实人常常**看不见**机会成本（这正是行为经济学补充的洞察）。
- [[沉没成本]]：理论上该忽略，真实人却常常掉进"沉没成本谬误"。

> [!tip] 关系定位
> 把古典模型当作**第一近似**（一阶逼近），行为经济学提供**系统性修正项**。工程上常用的"先近似、再修正"思路，正好对应这两者。

## 三、何时用哪一套

| 场景 | 更趁手的视角 |
| --- | --- |
| 市场总体趋势、长期均衡 | 古典理性人（个体偏差会被市场部分抹平） |
| 单次、个体、高情绪的决策 | 行为经济学 |
| 产品定价、菜单设计、助推 | 行为经济学（利用 [[框架效应]]、[[相对性]]、[[零价格效应]]） |
| 政策大盘、资源配置 | 古典 + 行为修正 |

> [!warning] 保持中立
> 行为经济学不是"古典经济学错了"，而是"古典经济学的适用边界在哪、超出边界后该怎么修正"。两者合起来，才是更完整的人。

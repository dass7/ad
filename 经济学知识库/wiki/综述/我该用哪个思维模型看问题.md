# 我该用哪个思维模型看问题

> [!note] 芒格的提醒
> "手里拿着锤子的人，看什么都像钉子。"——单一模型会扭曲判断。这页按**"什么场景用什么模型"**组织，帮你在面对具体问题时，调出对的工具，并尽量**用多个模型交叉验证**同一个问题。

## 场景速查表

| 你正面对的处境 | 调出的思维模型 |
| --- | --- |
| 要做取舍 | [[机会成本]] · [[边际分析]] · [[沉没成本]] |
| 不确定自己懂不懂 | [[能力圈]] |
| 想避免犯错 | [[逆向思考]] · [[人类误判心理学]] |
| 着眼长期 | [[复利]] |
| 面对复杂系统 | [[系统思维]] · [[因果分析]] |
| 要做重要决策 | [[决策框架]] · [[极度求真与极度透明]] · [[痛苦+反思=进步]] |
| 警惕自己被骗 | [[损失厌恶]] · [[锚定效应]] · [[过度自信]] · [[框架效应]] |
| 多个因素叠加 | [[Lollapalooza效应]] · [[多元思维模型]] |

## 一、做取舍时
真正的成本不是花了多少钱，而是放弃了什么。
- [[机会成本]]：每个选择的代价，是被放弃的最高价值选项。
- [[边际分析]]：决策看"多一单位"的得失，而非平均或总量。
- [[沉没成本]]：已花出且收不回的，不应影响当下决策——"别为打翻的牛奶哭泣"。

## 二、判断能力边界
- [[能力圈]]：知道自己知道什么、更要知道自己不知道什么。圈子大小不重要，知道边界在哪最重要。

## 三、避免犯错（反过来想）
- [[逆向思考]]：与其问"怎么成功"，先问"怎么会失败"，然后避开。
- [[人类误判心理学]]：系统盘点导致误判的心理倾向清单，是一份"错误检查表"。

## 四、追求长期复利
- [[复利]]：时间是朋友。小的优势持续叠加，会产生指数级结果——适用于财富、知识与关系。

## 五、理解系统如何运转
- [[系统思维]]：看要素之间的联系、反馈回路与时滞，而非孤立的点。
- [[因果分析]]：区分相关与因果，警惕"事后归因"。

## 六、走一套靠谱的决策流程
- [[决策框架]]：把决策结构化，减少情绪与偏差干扰。
- [[极度求真与极度透明]]：达利欧式原则，让真相浮出水面。
- [[痛苦+反思=进步]]：把每一次痛苦经历转化为可复用的原则。

## 七、警惕认知偏差
做任何判断前，先用这些"红灯"自检：
- [[损失厌恶]]：是不是因为怕损失而过度保守？
- [[锚定效应]]：第一个看到的数字是否绑架了我的估值？
- [[过度自信]]：我是不是高估了自己的判断？
- [[框架效应]]：换个说法，我的结论还成立吗？

## 八、多因素叠加（最强也最危险）
- [[Lollapalooza效应]]：多个模型/偏差**同向叠加**，产生远超单项之和的巨大效应（既能造就奇迹，也能酿成灾难）。
- [[多元思维模型]]：把上面所有模型组成"思维格栅"，遇事多角度交叉验证。

> [!tip] 用法心法
> 1. 先识别场景，调出表里对应的模型；
> 2. **至少用两个不同学科的模型**看同一问题；
> 3. 用 [[逆向思考]] 反着检验一遍；
> 4. 留意是否出现 [[Lollapalooza效应]] 的叠加放大。

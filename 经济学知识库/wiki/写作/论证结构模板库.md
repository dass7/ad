> [!summary] 一句话
> 写东西时不必从零想结构——从下面**挑一个骨架 + 套几个零件**即可。你强在画面与真诚，这页只补你缺的"逻辑骨架"。

> [!note] 怎么用
> ①先选一个**整体骨架**（A 区）；②再从**零件库**（B 区）取开头、论点句、论证段、反例、收尾；③写完用**自检清单**（C 区）过一遍。已写成的样板见 [[范文-我在表演努力]]（说理）与 [[范文2-退场券]]（叙事）。

## A. 五种整体骨架（按文体选）
1. **说理 march（议论文）**：钩子 → 反直觉论点 → 论证一/二/三 → 反例边界 → 落地。
   - 样板：[[范文-我在表演努力]]。适合：观点输出、书评、思辨。
2. **物象贯穿（抒情散文）**：一个意象开场 → 把它提炼成概念 → 用回忆当证据 → 反证 → 升华 → 回扣意象。
   - 样板：[[范文2-退场券]]。适合：随笔、个人叙事。
3. **对比结构**：A 是什么 / B 是什么 → 三维度对照 → 它们其实互补或谁更适用。
   - 样板：[[古典理性人vs行为经济学]]。适合：辨析两个概念/立场。
4. **成长时间线**：过去的我（具体场景）→ 转折事件 → 现在的我 → 我学到的那条原则。
   - 适合：申请文书、自我剖析。
5. **问题—诊断—方案（达利欧式）**：现象 → 真问题是什么 → 根因（[[因果分析]]）→ 可执行方案。
   - 呼应 [[决策框架]]。适合：复盘、方法论文章。

## B. 零件库（即取即用）
### 开头（钩子）
- **场景白描**：直接给一个画面，不解释（"那个篮球砸在太阳穴上的瞬间……"）。
- **反常识断言**："大多数人以为努力是好事，但有一种努力越多越糟。"
- **一个具体数字/细节**：以小见大。

### 亮论点的句式
- **"我以为我在 X，其实我在 Y。"**（制造张力，最好用）
- **"问题不在 A，而在 B。"**
- **"X 看起来是 ___，本质却是 ___。"**

### 论证段（把"感受"升级成"说理"）
- **感受 +「所以」**：写完一句感受，强迫补一句"所以——（结论）"。这是你最该练的一招。
- **跨学科三段式**：同一论点，用经济学一句 + 心理学一句 + 哲学/社会学一句各自印证。弹药库见 [[跨学科交叉验证]]。一个论点三科背书 = 说服力翻倍。
- **机制解释**：不止说"是什么"，要说"为什么会这样"（用 [[因果分析]]）。

### 反例 / 边界（让你显得不偏激、有分寸）
- **"是不是所有 X 都这样？当然不是——区别在 ___。"**
- 主动给出**对立面成立的条件**。这一步是新手与老手的分水岭。

### 收尾
- **回扣开头意象**（首尾同一个画面，文章立刻"圆"了）。
- **落到一个可执行的小动作**，而非空喊口号。
- **留白/反问**，把思考交还读者。

## C. 写完自检清单（类比 [[检查清单]]）
- [ ] 每段有没有一个**明确论点**（不是只有感受）？
- [ ] 最强的那个论点，有没有**第二、第三个学科**来背书？
- [ ] 有没有至少**一个反例/边界**？
- [ ] 长句是不是该**砍成短句**？（朗读一遍最准）
- [ ] 结尾是**回扣 + 落地**，还是空喊"我要变好"？
- [ ] 有没有**错别字/主谓断裂**？（通读一遍消掉大半）

> [!tip] 一句话心法
> 你已经有"感受"和"画面"——把每一处感受都接上"所以"，再借多学科各说一句，文章就从**散文**长出了**逻辑的骨头**。

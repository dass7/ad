# 🃏 闪卡模板（三种类型）

> 配合 Anki / Obsidian Anki Sync 使用。每张卡片末尾附关联双链。

## 类型一：概念卡（正反面）

```
正面：[一个具体的"为什么/是什么"问题]
背面：
- 要点1
- 要点2
- 促进/抑制因素
关联：[[相关知识点1]] [[相关知识点2]]
```

**示例：**
```
正面：铁的吸收率为什么差异如此大？
背面：
- 血红素铁：吸收率 15-35%（受影响小）
- 非血红素铁：吸收率 1-20%（受多因素影响）
- 促进因素：维生素C、肉类因子、胃酸
- 抑制因素：植酸、草酸、多酚、钙
关联：[[铁缺乏性贫血]] [[食物铁来源]]
```

## 类型二：公式卡

```
正面：[XX计算公式？]
背面：
公式 + 正常范围 + 判断标准
关联：[[营养评估]]
```

## 类型三：临床卡

```
正面：[具体临床情境问题]
背面：
推荐值 + 重点原则
关联：[[糖尿病营养管理]]
```

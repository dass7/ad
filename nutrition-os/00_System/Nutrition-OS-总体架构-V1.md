## 🔗 四、双链策略

> **哲学**：链接不是装饰，是思维的神经突触。

### 双链使用规则

**规则一：每个笔记至少 5 个双链**
```
[[概念]] → 链接相关知识点
[[MOC-xxx]] → 向上链接地图
[[人名/理论]] → 链接背景知识
[[案例-xxx]] → 链接实际案例
[[考点-xxx]] → 链接考试要点
```

**规则二：四种链接类型**

| 链接类型 | 格式示例 | 使用场景 |
|---------|---------|---------|
| 直接链接 | `[[葡萄糖]]` | 明确相关概念 |
| 别名链接 | `[[葡萄糖\|血糖的主要形式]]` | 语境中自然嵌入 |
| 标题链接 | `[[葡萄糖#代谢途径]]` | 指向具体章节 |
| 块链接 | `[[葡萄糖^abc123]]` | 引用具体段落 |

**规则三：双链思维导图**
```
每个知识点的链接方向：
  ↑ 向上：链接到 MOC（所属领域）
  ↓ 向下：链接到子概念（更细节）
  ← 向左：链接到先修知识（依赖关系）
  → 向右：链接到应用（临床/考试）
  ↔ 横向：链接到同级相关知识
```

### 高价值双链示例

```markdown
## 葡萄糖 → 应该双链到

上游基础：
- [[碳水化合物分类]] ← 葡萄糖是单糖
- [[消化吸收机制]] ← 如何被吸收

同级关联：
- [[果糖]] [[半乳糖]] ← 同为单糖
- [[胰岛素]] ← 调节葡萄糖代谢
- [[糖原]] ← 储存形式

下游应用：
- [[糖尿病营养管理]] ← 临床应用
- [[血糖生成指数GI]] ← 衍生概念
- [[糖酵解]] ← 代谢途径

考证链接：
- [[考点-碳水化合物]] ← 考试集合
```

## 📝 五、命名规范

> **黄金法则**：文件名 = 你搜索时会输入的词

### 命名格式规范

```
# 知识点笔记
格式：[核心词]-[修饰词].md
示例：维生素C-化学性质.md
     葡萄糖-代谢途径.md
     铁-吸收影响因素.md

# MOC 地图
格式：MOC-[领域名].md
示例：MOC-宏量营养素.md

# 每日笔记
格式：YYYY-MM-DD.md
示例：2026-05-27.md

# 周复习
格式：Week-YYYY-WXX.md
示例：Week-2026-W22.md

# 案例笔记
格式：Case-[疾病]-[编号].md
示例：Case-糖尿病-001.md

# 考题笔记
格式：Exam-[级别]-[题目关键词].md
示例：Exam-L3-蛋白质互补.md

# 错题笔记
格式：Wrong-[日期]-[知识点].md
示例：Wrong-2026-05-27-脂溶性维生素.md
```

### 严禁的命名方式

```
❌ 笔记1.md          → 无意义
❌ 今天学的.md        → 无法检索
❌ 维C.md            → 缩写不规范
❌ VitaminC.md       → 中英混乱（统一用中文）
❌ 维生素C(1).md      → 不允许数字后缀（用修饰词区分）
```

## 🃏 六、卡片笔记规范

> **原子化原则**：每张卡片只回答一个问题。

### 原子笔记 YAML 模板

```yaml
---
title: "铁的吸收影响因素"
type: concept
domain: [micronutrient, biochemistry]
importance: ★★★★★
exam_level: [level3, level4]
exam_freq: 高频
created: 2026-05-27
updated: 2026-05-27
status: developing
tags:
  - type/concept
  - domain/micronutrient
  - exam/highfreq
  - importance/★★★★★
review_count: 0
next_review: 2026-05-30
source: "《中国居民膳食营养素参考摄入量》"
related:
  - "[[维生素C]]"
  - "[[血红素铁]]"
  - "[[植酸]]"
  - "[[贫血营养治疗]]"
---
```

### 卡片内容规范（6C原则）

```
C1 - Core（核心定义）：1-3句话，不超过50字
C2 - Context（背景）：为什么重要
C3 - Connection（连接）：与哪些知识相关
C4 - Clinical（临床）：实际应用
C5 - Cram（记忆）：记忆口诀/类比
C6 - Check（考点）：高频考试点
```

### 笔记大小标准

```
✅ 合适：200-800字（一个屏幕内）
⚠️ 太短：< 100字（可能太空洞）
⚠️ 太长：> 1500字（需要拆分）
❌ 超长：> 3000字（必须原子化拆分）
```

```

  → 把零散笔记连接到 MOC

  → 重新学习标记为"模糊"的知识点

步骤五（5min）：下周计划
  → 根据24个月路线图确认下周目标
  → 在 Home.md 更新进度
```

## 🧮 九、间隔复习系统

### Anki 闪卡规范

**卡片类型一：概念卡（正反面）**
```
正面：铁的吸收率为什么差异如此大？
背面：
- 血红素铁：吸收率 15-35%（受影响小）
- 非血红素铁：吸收率 1-20%（受多因素影响）
- 促进因素：[[维生素C]]、[[肉类因子]]、胃酸
- 抑制因素：[[植酸]]、[[草酸]]、[[多酚]]、钙
关联：[[铁缺乏性贫血]] [[食物铁来源]]
```

**卡片类型二：公式卡**
```
正面：BMI计算公式？
背面：
BMI = 体重(kg) ÷ 身高²(m²)
正常范围：18.5-23.9（中国标准）
超重：24-27.9
肥胖：≥28
关联：[[营养评估]] [[体重管理]]
```

**卡片类型三：临床卡**
```
正面：2型糖尿病患者，碳水化合物应占总能量多少？
背面：
推荐：45-60%（不低于130g/天）
重点：选低GI食物，均匀分配
关联：[[糖尿病营养管理]] [[GI值]] [[碳水化合物]]
```

```markdown
- BMI：29.7 → [[肥胖判断标准]]

1. 能量摄入超标，导致肥胖
```

```
### 饮食原则
1. 低GI饮食：[[GI值表]]

3. 增加膳食纤维：目标 25-30g/天

5. 限制精制碳水和添加糖
```

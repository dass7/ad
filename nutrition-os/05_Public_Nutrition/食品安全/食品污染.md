# 食品污染

## 一句话
> 食品在生产到食用过程中混入有害因素，分生物性、化学性、物理性。

## 核心要点
| 类 | 例 |
|---|---|
| 生物性 | 细菌、霉菌毒素(黄曲霉)、寄生虫 |
| 化学性 | 农药残留、重金属、亚硝酸盐、多环芳烃 |
| 物理性 | 异物、放射性 |

- 黄曲霉毒素：强致癌，霉变粮油

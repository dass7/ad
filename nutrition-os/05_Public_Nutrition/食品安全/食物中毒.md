# 食物中毒

## 一句话
> 摄入含致病因素的食物引起的急性/亚急性疾病；分细菌性、真菌性、化学性、有毒动植物性。

## 分类与代表（⚠️核对）
| 类 | 代表 |
|---|---|
| 细菌性（最常见） | 沙门氏菌、副溶血性弧菌、金黄色葡萄球菌肠毒素、肉毒梭菌 |
| 真菌毒素 | 黄曲霉毒素（强致癌，霉变花生/玉米） |
| 有毒动植物 | 河豚、毒蘑菇、四季豆（皂苷）、发芽马铃薯（龙葵素） |
| 化学性 | 亚硝酸盐、农药、重金属 |

- 特点：潜伏期短、集体发病、与进食相关、不传染

# 中国居民平衡膳食宝塔

## 一句话
> 把膳食指南可视化为五层宝塔，按层级给出每日各类食物推荐量。

## 五层（自下而上，⚠️核对克数）
1. 谷薯类（最底、最多）
2. 蔬菜、水果
3. 鱼禽肉蛋（动物性食物）
4. 奶、大豆及坚果
5. 油、盐（塔尖、最少）
- 另：每日足量饮水、身体活动

## 🎯 高频考点
- [ ] 五层结构与顺序（谷薯在底、油盐在尖）
- [ ] 各层每日推荐量（高频填空）
- [ ] 盐<5g、油 25–30g 等关键值 ⚠️核对

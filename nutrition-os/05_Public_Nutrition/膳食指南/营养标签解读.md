# 营养标签解读

## 一句话
> 预包装食品强制标示营养成分表，帮助消费者按需选择。

## 核心要点（⚠️核对法规）
- **强制标示"1+4"**：能量 + 蛋白质、脂肪、碳水化合物、**钠**
- **NRV%**：营养素参考值百分比，看一份贡献多少日需
- 配料表按用量**递减**排列；警惕"氢化油"=[[反式脂肪酸]]
- 声称规则：如"低钠""无糖"有量化门槛

## 🎯 高频考点
- [ ] 强制标示 1+4（含钠）
- [ ] NRV% 含义
- [ ] 配料表递减排列

# 骨质疏松营养

## 一句话
> 以充足**钙 + 维生素D + 蛋白质**配合负重运动，维护骨密度。

## 核心要点（⚠️核对）
- [[钙]]：奶制品为佳，必要时补钙
- [[维生素D]]：促钙吸收，日照 + 必要补充
- 适量优质蛋白；充足蔬果（钾、镁、维K）
- 限：过量钠、咖啡因、酒、磷酸饮料（促钙流失）
- 配合负重/抗阻运动；老年防跌倒

## 🎯 高频考点
- [ ] 钙 + 维D 是核心
- [ ] 限钠/咖啡因/碳酸饮料
- [ ] 营养 + 负重运动联合

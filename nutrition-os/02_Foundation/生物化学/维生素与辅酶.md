# 维生素与辅酶

## 一句话
> 多种 B 族维生素是酶发挥作用所需的**辅酶/辅基**，缺乏直接拖垮代谢。

## 核心对应（⚠️核对）
| 维生素 | 辅酶形式 | 参与 |
|---|---|---|
| B1 硫胺素 | TPP | 糖代谢脱羧（丙酮酸→乙酰CoA） |
| B2 核黄素 | FAD/FMN | 氧化还原（FADH₂） |
| B3 烟酸 | NAD/NADP | 氧化还原（NADH） |
| B6 吡哆醇 | PLP | 氨基酸转氨 |
| B9 叶酸 | THF | 一碳单位转移 |
| B12 钴胺素 | — | 与叶酸协同、神经 |

## 🎯 高频考点
- [ ] B1→TPP→糖代谢；B2→FAD；B3→NAD
- [ ] B6→PLP→氨基酸代谢
- [ ] 叶酸→一碳单位

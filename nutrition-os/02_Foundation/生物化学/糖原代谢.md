# 糖原代谢

## 一句话
> 糖原是葡萄糖的**储存形式**；肝糖原调血糖，肌糖原供肌肉自用。

## 核心要点
| | 肝糖原 | 肌糖原 |
|---|---|---|
| 作用 | 维持**血糖**（可释放葡萄糖入血） | 仅供**肌肉**自身收缩 |
| 原因 | 肝有葡萄糖-6-磷酸酶 | 肌肉缺该酶，不能放糖入血 |

- 合成（糖原合成）↔ 分解（糖原分解）
- 激素调节：[[胰岛素]]促合成，[[胰高血糖素]]/肾上腺素促分解

# 官能团

## 一句话
> 决定有机分子化学性质的特定原子团；营养分子由其性质决定行为。

## 核心要点
| 官能团 | 结构 | 出现在 |
|---|---|---|
| 羟基 | —OH | 糖、醇 |
| 羧基 | —COOH | 脂肪酸、氨基酸（酸性） |
| 氨基 | —NH₂ | 氨基酸（碱性） |
| 羰基 | C=O | 醛糖/酮糖 |
| 酯键/肽键 | —COO—／—CONH— | 甘油三酯 / 蛋白质 |

## 🎯 高频考点
- [ ] 羧基酸性、氨基碱性
- [ ] 氨基酸同时含 —COOH 与 —NH₂（两性）
- [ ] 肽键/酯键的形成

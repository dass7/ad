# 脂类化学

## 一句话
> 脂类 = 不溶于水、溶于有机溶剂的一类物质，含甘油三酯、磷脂、固醇等。

## 分类
| 类 | 组成/例 | 作用 |
|---|---|---|
| 甘油三酯 | 甘油 + 3 脂肪酸 | 储能主体（9 kcal/g） |
| 磷脂 | 甘油+脂肪酸+磷酸+含氮碱 | 构成细胞膜 |
| 固醇 | 胆固醇 | 膜成分、合成激素/胆汁酸/维D前体 |

- 脂肪酸：饱和 / 单不饱和 / 多不饱和；详见 [[脂肪酸分类]]、[[必需脂肪酸]]

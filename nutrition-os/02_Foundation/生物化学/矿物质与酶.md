# 矿物质与酶

## 一句话
> 许多矿物质作为酶的辅因子或激活剂参与代谢。

## 核心对应（⚠️核对）
| 矿物质 | 相关酶/功能 |
|---|---|
| 锌 | 碳酸酐酶、多种脱氢酶 |
| 硒 | 谷胱甘肽过氧化物酶（抗氧化） |
| 铁 | 细胞色素、过氧化氢酶 |
| 铜 | 超氧化物歧化酶、铜蓝蛋白 |
| 镁 | ATP 相关激酶辅因子 |
| 锰/钼 | 多种氧化酶 |

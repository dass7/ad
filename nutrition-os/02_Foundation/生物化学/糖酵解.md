# 糖酵解

## 一句话
> 葡萄糖在**细胞质**中分解为丙酮酸的过程，**不需氧**，是供能的第一步。

## 核心要点
- 部位：细胞质（胞浆）
- 终产物：1 分子葡萄糖 → 2 丙酮酸
- 净产能：**2 ATP + 2 NADH**（⚠️核对）
- 关键限速酶：己糖激酶、**磷酸果糖激酶-1（PFK-1，最关键）**、丙酮酸激酶
- 去向：
  - 有氧 → 丙酮酸入线粒体 → [[三羧酸循环]]
  - 缺氧 → 丙酮酸还原为**乳酸**（如剧烈运动肌肉）

## 🎯 高频考点
- [ ] 部位在细胞质、不耗氧
- [ ] 净生成 2 ATP
- [ ] 限速酶 PFK-1
- [ ] 缺氧产物=乳酸

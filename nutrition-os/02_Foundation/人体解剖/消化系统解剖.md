# 消化系统解剖

## 一句话
> 消化系统 = 消化道（口→肛）+ 消化腺（肝/胰/唾液腺等）。

## 核心结构
- **消化道**：口腔 → 咽 → 食道 → 胃 → 小肠（十二指肠/空肠/回肠）→ 大肠 → 直肠 → 肛门
- **消化腺**：
  - 唾液腺（唾液淀粉酶）
  - 肝（分泌**胆汁**，乳化脂肪）/ 胆囊（储存浓缩胆汁）
  - 胰（外分泌消化酶 + 内分泌[[胰岛素]]/[[胰高血糖素]]）
- 小肠特征：**绒毛+微绒毛**极大扩展吸收面积

- [ ] 肝分泌胆汁、胆囊储存

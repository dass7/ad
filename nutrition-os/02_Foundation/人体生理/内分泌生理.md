# 内分泌生理

## 一句话
> 激素经血液调节代谢、生长与稳态；与营养代谢关系最密切的是胰岛、甲状腺、肾上腺。

## 核心要点（与营养相关）
| 激素 | 来源 | 营养相关作用 |
|---|---|---|
| [[胰岛素]] | 胰 β 细胞 | 降血糖、促合成储能 |
| [[胰高血糖素]] | 胰 α 细胞 | 升血糖、动员 |
| 甲状腺激素 T3/T4 | 甲状腺（需[[碘]]） | 提高基础代谢率 |
| 皮质醇 | 肾上腺皮质 | 升糖、促分解（应激） |
| 生长激素 | 垂体 | 促蛋白合成、生长 |

## 🎯 高频考点
- [ ] 胰岛素/胰高血糖素拮抗调血糖
- [ ] 甲状腺激素调基础代谢（与碘）
- [ ] 皮质醇升糖促分解

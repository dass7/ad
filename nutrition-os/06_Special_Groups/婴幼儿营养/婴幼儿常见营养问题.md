# 婴幼儿常见营养问题

## 一句话
> 高发：维生素D缺乏性佝偻病、缺铁性贫血、生长迟缓、食物过敏。

## 核心要点
- **佝偻病**：[[维生素D]] 缺乏 + 钙磷代谢异常 → 补 D + 日照
- **缺铁性贫血**：6 月后母乳铁不足 → [[辅食添加原则|首选富铁辅食]]
- 食物过敏：逐一引入新食物观察
- 喂养不当致超重或消瘦

## 🎯 高频考点
- [ ] 佝偻病（维D缺）与缺铁性贫血两大问题
- [ ] 预防要点（补D、富铁辅食）

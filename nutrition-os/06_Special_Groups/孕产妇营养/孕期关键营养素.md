# 孕期关键营养素

## 一句话
> 孕期需重点保障：叶酸、铁、碘、钙、DHA 等，关系胎儿发育。

## 核心要点（⚠️核对剂量）
| 营养素 | 关键意义 |
|---|---|
| [[叶酸]] | 孕前 3 月起补，防**神经管畸形** |
| [[铁]] | 防孕期[[铁缺乏与贫血\|缺铁性贫血]]，血容量增 |
| [[碘]] | 防胎儿克汀病、智力损害 |
| [[钙]] | 胎儿骨骼，预防孕妇骨丢失 |
| DHA | 胎儿脑与视网膜发育 |

- 能量与蛋白质随孕中晚期递增；控制体重适度增长

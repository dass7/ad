# 减脂增肌饮食策略

## 一句话
> 减脂靠能量负平衡，增肌靠足量蛋白 + 抗阻训练 + 适度能量盈余。

## 核心要点（⚠️核对数值）
- **减脂**：温和能量赤字、保高蛋白防肌肉流失、控精制碳水
- **增肌**：能量略盈余、蛋白质约 1.6–2.0 g/kg/天、训练后补碳水+蛋白 ⚠️核对
- 碳水安排在训练前后；脂肪不过低
- 个体化 + 循序渐进，避免极端

## 🎯 高频考点
- [ ] 减脂=负平衡、增肌=正平衡+高蛋白
- [ ] 运动人群蛋白需求高于一般成人
- [ ] 训练后营养时机

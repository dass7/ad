# 🎓 MOC-导师课程

> **这个系列的目标不是让你背知识，而是建立营养学底层思维。**
>
> 每一课都从本质出发，连接人体系统、跨学科视角、临床应用、考试考点与未来趋势。
> 读完这个系列，你将拥有一个营养学家的思维方式，而不只是一本知识手册。

---

## 📐 导师教学框架（每课包含）

```
1.  本质          → 这个知识在说什么（第一原理）
2.  起源          → 为什么会出现
3.  功能          → 它解决什么问题
4.  系统关系      → 与人体整体的连接
5.  跨学科        → 物理/化学/心理/经济学视角
6.  考试考点      → 四/三/二级高频考点
7.  临床应用      → 在医院/诊所如何决策
8.  现实应用      → 给普通人的建议
9.  伪科学辨别    → 常见错误说法
10. 前沿争议      → 科学界尚未解决的问题
11. 过时内容      → 教材已经落后的地方
12. 未来方向      → 值得重点投资的方向
13. AI影响        → 这个知识在AI时代的价值变化
```

## 🧠 导师教学哲学

> **"学营养学，不是为了成为一本会走路的营养成分表。"**

真正的营养师能力体现在：
1. **当信息缺失时**，能用系统思维推理出合理方向
2. **当证据冲突时**，能判断哪个更可信
3. **当面对患者时**，能把科学翻译成行动
4. **当行业变化时**，能快速整合新知识

这个系列的每一课，都在训练这四种能力。

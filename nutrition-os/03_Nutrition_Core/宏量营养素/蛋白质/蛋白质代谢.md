# 蛋白质代谢

## 一句话
> 蛋白质不断合成与降解（周转）；氨基酸可供能、转化或合成新蛋白。

## 核心要点
- 氨基酸去路：合成蛋白、生成含氮化合物、脱氨供能/[[糖异生]]
- 脱氨基 → 氨 → **尿素循环**（肝）→ 尿排出
- 氨基酸池概念；与[[氮平衡]]关联

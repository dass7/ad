# 蛋白质食物来源与推荐量

## 一句话
> 一般成人约 0.8–1.0 g/kg/天，优质蛋白应占一定比例。

## 核心要点（⚠️核对）
- 推荐量：成人约 **0.8–1.0 g/kg/天**（供能比 10%–20%）
- 优质蛋白来源：蛋、奶、鱼、瘦肉、大豆
- 善用[[蛋白质互补]]（谷+豆）
- 特殊状态（孕、运动、老年、疾病）需求不同

# 蛋白质营养价值评价

## 一句话
> 从「量、质（氨基酸模式）、消化利用率」三方面评价蛋白质好坏。

## 核心指标
| 指标 | 含义 |
|---|---|
| 含量 | 食物中蛋白质多少 |
| 氨基酸评分 AAS | 待测蛋白与理想模式的最缺氨基酸之比 → 决定**限制氨基酸** |
| PDCAAS | AAS × 消化率（经典常用） |
| DIAAS | 更新指标，按回肠真消化率 |
| 生物价 BV | 储留氮/吸收氮 |
| 蛋白质功效比 PER | 动物增重法 |

- 优质蛋白：氨基酸模式接近人体（蛋、奶、肉、大豆）

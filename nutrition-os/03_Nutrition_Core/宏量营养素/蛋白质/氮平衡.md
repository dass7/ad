# 氮平衡

## 一句话
> 比较摄入氮与排出氮，反映体内蛋白质合成与分解的状态。

## 核心要点
- 氮平衡 = 摄入氮 −（尿氮 + 粪氮 + 皮肤等损失）
- **零平衡**：摄入=排出（健康成人）
- **正平衡**：摄入>排出 → 生长、孕妇、康复、健身增肌
- **负平衡**：摄入<排出 → 饥饿、消耗性疾病、烧伤、老年肌肉流失
- 蛋白质含氮约 16% → 氮 ×6.25 ≈ 蛋白质

# 蛋白质互补

## 一句话
> 把限制氨基酸不同的食物搭配吃，互相补足，提高整体蛋白质利用率。

## 核心要点
- 原理：A 缺的氨基酸由 B 补，反之亦然
- 经典例：**谷类（缺赖氨酸）+ 豆类（缺蛋氨酸）** → 互补（如米饭+豆制品）
- 三原则（⚠️核对）：食物种属越远越好、搭配种类越多越好、同餐食用

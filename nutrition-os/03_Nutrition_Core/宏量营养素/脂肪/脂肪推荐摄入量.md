# 脂肪推荐摄入量

## 一句话
> 总脂肪约 20%–30% 供能，重在结构（限饱和/反式、增不饱和）。

## 核心要点（⚠️核对）
- 总脂肪：约 **20%–30%** 总能量
- 饱和脂肪：<10%（部分建议更低）
- [[反式脂肪酸]]：<1%
- 增 n-3/n-6 等不饱和（见[[脂肪酸分类]]）

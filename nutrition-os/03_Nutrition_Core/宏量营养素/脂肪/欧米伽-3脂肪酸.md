# 欧米伽-3脂肪酸

## 一句话
> n-3 系多不饱和脂肪酸（α-亚麻酸、EPA、DHA），抗炎、护心血管、益脑。

## 核心要点
- 成员：ALA（植物，亚麻籽/核桃）、**EPA/DHA**（深海鱼）
- 功能：降甘油三酯、抗炎、DHA 促脑/视网膜发育
- ALA→EPA/DHA 转化率低 → 直接吃鱼或鱼油更有效
- 与 n-6 需适当比例

## 🎯 高频考点
- [ ] 三种 n-3 及来源
- [ ] EPA/DHA 来自深海鱼
- [ ] 心血管与神经发育作用

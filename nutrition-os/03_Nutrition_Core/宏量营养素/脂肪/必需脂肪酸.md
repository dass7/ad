# 必需脂肪酸

## 一句话
> 人体不能自身合成、必须由食物供给的脂肪酸——**亚油酸(n-6)** 与 **α-亚麻酸(n-3)**。

## 核心要点
- 两种必需脂肪酸：亚油酸（n-6）、α-亚麻酸（n-3）
- 衍生：α-亚麻酸→EPA/DHA；亚油酸→花生四烯酸（⚠️转化率有限，常需直接补 EPA/DHA）
- 功能：构成细胞膜磷脂、合成前列腺素等；缺乏→皮肤损害、生长迟缓
- 来源：植物油（亚油酸）、深海鱼/亚麻籽（n-3）

# 碳水化合物食物来源与推荐量

## 一句话
> 以全谷薯类为主，供能比约 50%–65%。

## 核心要点（⚠️核对）
- 供能比：约 **50%–65%** 总能量（AMDR）
- 优选：全谷、薯类、杂豆；限精制糖（添加糖 <10%、最好 <5% 供能）
- 来源：米面、薯类、水果、奶

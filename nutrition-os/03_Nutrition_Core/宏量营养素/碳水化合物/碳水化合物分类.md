# 碳水化合物分类

## 一句话
> 按聚合度分单糖、双糖、寡糖、多糖；营养上重点区分「可吸收糖」与「膳食纤维」。

## 核心要点
| 类 | 例 | 营养意义 |
|---|---|---|
| 单糖 | 葡萄糖、果糖、半乳糖 | 直接吸收，升糖快 |
| 双糖 | 蔗糖、乳糖、麦芽糖 | 需双糖酶分解 |
| 寡糖 | 低聚果糖/低聚半乳糖 | 益生元，部分人胀气 |
| 多糖-可消化 | 淀粉、糊精 | 主要能量来源 |
| 多糖-不可消化 | [[膳食纤维]]（纤维素/果胶等） | 不供能、调肠道血糖血脂 |

- 乳糖不耐受：缺乳糖酶 → 乳糖到结肠发酵产气

# 碳水化合物消化吸收

## 一句话
> 淀粉经淀粉酶→双糖→单糖，在小肠以单糖形式吸收入血。

## 核心要点
- 口腔/胰淀粉酶分解淀粉 → 双糖
- 小肠刷状缘双糖酶（麦芽糖酶/蔗糖酶/乳糖酶）→ 单糖
- 吸收：[[葡萄糖]]/半乳糖主动转运(SGLT1)、果糖易化扩散
- 乳糖酶缺乏 → 乳糖不耐受

## 🎯 高频考点
- [ ] 消化为单糖才吸收
- [ ] 葡萄糖主动转运、果糖易化扩散
- [ ] 乳糖不耐受机制

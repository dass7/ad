# 各营养素吸收机制

## 一句话
> 营养素主要在小肠吸收，方式有主动转运、易化扩散、被动扩散。

## 核心要点（⚠️核对）
| 营养素 | 吸收形式/方式 |
|---|---|
| 葡萄糖/半乳糖 | 主动转运（Na⁺依赖，SGLT1） |
| 果糖 | 易化扩散 |
| 氨基酸 | 主动转运（多种载体） |
| 脂肪 | 胆汁乳化→微团→淋巴（乳糜微粒） |
| 钙 | 主动（需[[维生素D]]）+ 被动 |
| 铁 | 二价更易吸收，受[[维生素C]]促进 |

## 🎯 高频考点
- [ ] 葡萄糖/氨基酸=主动转运，果糖=易化扩散
- [ ] 脂肪经淋巴入血
- [ ] 钙铁吸收的促进因素

# 基础代谢BMR

## 一句话
> 清醒、空腹、安静、适温下维持基本生命活动的最低能量消耗。

## 核心要点（⚠️核对公式）
- 占总能量消耗最大（60%–70%）
- 影响因素：体表面积/瘦体重（最主要）、年龄(↓)、性别(男>女)、甲状腺激素、环境温度
- 估算公式：Mifflin-St Jeor、Harris-Benedict 等
- REE（静息能量消耗）≈ BMR，临床更常测

## 🎯 高频考点
- [ ] BMR 定义与测定条件（清醒/空腹/静卧/适温）
- [ ] 主要影响因素（瘦体重、年龄、性别）
- [ ] 占总能耗比例最大

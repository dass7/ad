# 能量平衡与体重

## 一句话
> 摄入与消耗的差额决定体重：正平衡增重、负平衡减重。

## 核心要点
- 正平衡 → 体重增加（脂肪储存）
- 负平衡 → 体重下降（动员储能）
- 约 7700 kcal ≈ 1 kg 体脂（估算，⚠️核对）
- 长期能量平衡 = 稳定体重

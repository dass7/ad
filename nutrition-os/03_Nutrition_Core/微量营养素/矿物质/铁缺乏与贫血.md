# 铁缺乏与贫血

## 一句话
> 缺铁性贫血（IDA）= 体内铁不足致血红蛋白合成下降，呈**小细胞低色素**性贫血。

## 核心要点
- 三阶段：铁减少期 → 红细胞生成缺铁期 → 缺铁性贫血期
- 高危：婴幼儿、孕妇、育龄女性、青少年
- 表现：乏力、面色苍白、头晕、指甲变脆、注意力下降
- 营养干预：
  - 多吃血红素铁（红肉/动物肝/血）
  - 搭配 [[维生素C]] 促非血红素铁吸收
  - 避免随餐浓茶/咖啡

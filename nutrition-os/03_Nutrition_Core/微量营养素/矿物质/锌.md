# 锌

## 一句话
> 多种酶的组成/激活成分，关系生长、免疫、味觉与伤口愈合。

## 核心要点
- 功能：酶辅因子、促生长发育、维持味觉、免疫、伤口愈合
- **缺乏**：生长迟缓、食欲↓味觉↓、免疫力下降、伤口难愈、儿童多见
- 来源：贝类（牡蛎最高）、红肉、动物性食物吸收好；植酸抑制吸收

## 🎯 高频考点
- [ ] 缺乏：生长迟缓、味觉减退、食欲差
- [ ] 来源以贝类/红肉为佳
- [ ] 植酸抑制吸收

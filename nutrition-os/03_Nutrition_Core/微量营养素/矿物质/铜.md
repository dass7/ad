# 铜

## 一句话
> 多种氧化酶的成分，参与铁代谢、抗氧化与结缔组织形成。

## 核心要点
- 功能：铜蓝蛋白(促铁利用)、超氧化物歧化酶(抗氧化)、胶原交联
- 缺乏：贫血(影响铁利用)、骨骼异常（少见）
- 与锌竞争吸收（过量补锌可致缺铜）
- 来源：动物肝、贝类、坚果、全谷

## 🎯 高频考点
- [ ] 参与铁代谢（铜蓝蛋白）
- [ ] 锌铜竞争吸收

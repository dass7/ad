# 磷

## 一句话
> 体内含量第二的矿物质，与钙构成骨骼，参与 ATP、核酸、磷脂。

## 核心要点
- 功能：骨牙构成、能量(ATP)、核酸/磷脂、酸碱缓冲
- 分布广，普通膳食很少缺乏；加工食品含磷添加剂多
- **钙磷比**影响吸收；肾病患者需**限磷**（高磷血症）
- 来源：肉、蛋、奶、坚果、全谷

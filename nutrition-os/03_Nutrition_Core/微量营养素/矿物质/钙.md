# 钙

## 一句话
> 人体含量最多的矿物质，99% 在骨骼牙齿，参与凝血、肌肉收缩、神经传导。

## 核心要点
- 吸收促进：[[维生素D]]、乳糖、适量蛋白质
- 吸收抑制：植酸、草酸、过多膳食纤维、过多磷
- **缺乏**：儿童佝偻病、成人骨软化、老年骨质疏松
- 来源：奶及奶制品（最佳）、豆制品、深绿叶菜、虾皮

## 🎯 高频考点
- [ ] 含量最多的矿物质、99% 在骨牙
- [ ] 维生素D 促吸收；草酸/植酸抑制
- [ ] 奶制品是最佳来源
- [ ] 缺乏与骨骼疾病

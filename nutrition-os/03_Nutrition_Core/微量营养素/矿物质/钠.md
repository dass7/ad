# 钠

## 一句话
> 细胞外主要阳离子，调节体液渗透压与血压；摄入过多与高血压相关。

## 核心要点
- 功能：维持渗透压、酸碱平衡、神经肌肉兴奋
- **过量**：升血压风险（见[[心血管疾病营养]]）
- 食盐(NaCl)是主要来源；隐性钠（加工食品、酱油、咸菜）
- 控盐目标：成人食盐 **<5 g/天**（WHO）⚠️核对中国标准
- 钠钾比：增钾有助降压（见[[钾]]）

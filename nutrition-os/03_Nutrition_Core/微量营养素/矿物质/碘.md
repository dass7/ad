# 碘

## 一句话
> 合成甲状腺激素的必需元素；缺乏致甲状腺肿与克汀病。

## 核心要点
- 功能：构成甲状腺激素（T3/T4）→ 调节代谢、促生长发育
- **缺乏**：地方性甲状腺肿（大脖子病）；孕期缺碘→胎儿**克汀病**（智力/发育障碍）
- **过量**：也可致甲状腺功能异常（U 形风险）
- 来源：海带、紫菜、海产品；**碘盐**（公共营养干预）

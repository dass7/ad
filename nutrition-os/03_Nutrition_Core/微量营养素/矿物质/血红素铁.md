# 血红素铁

## 📌 概念
> 血红素铁是以卟啉环结合形式存在的铁，来自动物性食物的血红蛋白和肌红蛋白，整体吸收、效率高且基本不受膳食因素干扰。

## 🔬 本质：两种膳食铁的对比

| | 血红素铁 | 非血红素铁 |
|---|---------|-----------|
| 来源 | 红肉、动物血、肝脏、鱼禽 | 植物性食物、蛋、奶 |
| 吸收率 | **15-35%** | **1-20%** |
| 受膳食影响 | 很小 | 很大 |
| 促进因素 | — | [[维生素C]]、肉类因子、胃酸 |
| 抑制因素 | — | 植酸、草酸、多酚（茶/咖啡）、钙 |

## ❓ 为什么重要
- 解释了"同样含铁，菠菜补铁远不如猪肝"——补铁食谱设计的核心依据
- 铁吸收影响因素是四级/三级**最高频考点**之一

## 🔗 与其他系统关系
- 协同：→ [[维生素C]] 帮非血红素铁还原为 Fe²⁺
- 下游：→ [[铁缺乏性贫血]] 干预首选血红素铁丰富的食物
- 交叉：素食人群铁营养是 [[MOC-特殊人群营养]] 的实际问题

## ⚠️ 常见误区
1. ❌ 误区：菠菜是补铁明星
   ✅ 正确：菠菜铁为非血红素铁且富含草酸，实际可利用率很低
2. ❌ 误区：铁锅炒菜能有效补铁
   ✅ 正确：溶出的无机铁量少且吸收率低，不能替代膳食铁

## 🎯 高频考试点
- [ ] 两类铁的吸收率范围（15-35% vs 1-20%）
- [ ] 非血红素铁的促进/抑制因素各列 3 个
- [ ] 餐后立即喝浓茶为何影响补铁（多酚螯合）

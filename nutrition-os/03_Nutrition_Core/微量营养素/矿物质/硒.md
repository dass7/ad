# 硒

## 一句话
> 谷胱甘肽过氧化物酶的组成成分，抗氧化，与维生素E协同。

## 核心要点
- 功能：抗氧化（GSH-Px）、保护心肌、与[[维生素E]]协同
- **缺乏**：克山病（地方性心肌病）、大骨节病相关
- 过量：硒中毒（脱发、指甲变形）——安全窗较窄
- 来源：海产、动物内脏、巴西坚果；与土壤硒含量相关

## 🎯 高频考点
- [ ] 缺乏 → 克山病
- [ ] 抗氧化、与维E协同
- [ ] 安全窗窄、过量可中毒

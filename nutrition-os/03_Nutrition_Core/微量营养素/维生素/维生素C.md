# 维生素C

## 📌 概念
> 维生素C（抗坏血酸）是水溶性维生素，最强的水相抗氧化剂之一，参与胶原合成与铁吸收，缺乏导致坏血病。

## 🫀 在人体中的作用

| 功能 | 机制 | 缺乏后果 |
|------|------|---------|
| 胶原合成 | 脯氨酸/赖氨酸羟化的辅因子 | 坏血病（牙龈出血、伤口不愈） |
| 促进铁吸收 | 把 Fe³⁺ 还原为易吸收的 Fe²⁺ | 加重 [[铁缺乏性贫血]] |
| 抗氧化 | 清除自由基、再生维生素E | 氧化应激↑ |
| 免疫支持 | 白细胞高浓度富集 | 抵抗力下降 |

## ❓ 为什么重要
- "维生素C + 非血红素铁同餐"是营养配餐的经典操作，考试和实践都高频
- 水溶性维生素的代表：不储存、需每日摄入、过量随尿排出

## 🔗 与其他系统关系
- 协同：与 [[血红素铁]] 对比理解——非血红素铁需要 VC 帮忙，血红素铁不需要
- 下游：→ [[铁缺乏性贫血]] 的膳食干预必配 VC 丰富的果蔬
- 拮抗：高温、碱性环境、长时间储存破坏 VC（烹饪学交叉点）

## ⚠️ 常见误区
1. ❌ 误区：大剂量 VC 能防治感冒
   ✅ 正确：证据不支持预防作用，常规摄入量即可；长期大剂量增加肾结石风险
2. ❌ 误区：水果越酸 VC 越多
   ✅ 正确：酸味主要来自有机酸；鲜枣、猕猴桃、青椒才是 VC 大户

## 🎯 高频考试点
- [ ] 成人 RNI：100 mg/d；UL：2000 mg/d（以最新版中国 DRIs 为准）
- [ ] 缺乏病名称（坏血病）及典型体征
- [ ] 促进非血红素铁吸收的机制（Fe³⁺→Fe²⁺）

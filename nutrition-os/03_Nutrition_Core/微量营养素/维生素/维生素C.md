# 维生素C

## 一句话
> 水溶性维生素（抗坏血酸），强抗氧化，**缺乏致坏血病**。

## 核心要点
- 功能：促胶原合成、抗氧化、促**非血红素铁吸收**、参与免疫
- **缺乏**：坏血病（牙龈出血、伤口难愈、瘀点）
- 特性：易被热/氧/碱破坏（烹调损失大）
- 来源：新鲜蔬菜水果（鲜枣、柑橘、彩椒）

## 🎯 高频考点
- [ ] 缺乏 → 坏血病
- [ ] 促进非血红素铁吸收（与[[铁]]搭配）
- [ ] 促胶原合成、抗氧化
- [ ] 烹调易损失

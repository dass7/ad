# 维生素B12

## 一句话
> 钴胺素，参与造血与神经髓鞘；吸收依赖胃**内因子**，几乎仅存于动物性食物。

## 核心要点
- 功能：与[[叶酸]]协同造血、维持神经髓鞘
- 吸收：需胃壁细胞分泌的**内因子**（缺→恶性贫血）
- **缺乏**：巨幼细胞贫血 + **神经系统损害**（与单纯叶酸缺的区别点）
- 来源：肉、肝、蛋、奶；**纯素食者易缺**

## 🎯 高频考点
- [ ] 吸收需内因子
- [ ] 缺乏伴神经损害（区别叶酸缺）
- [ ] 纯素食人群风险

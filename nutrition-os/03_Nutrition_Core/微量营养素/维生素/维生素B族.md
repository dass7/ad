# 维生素B族

## 一句话
> 一组水溶性维生素，多作为辅酶参与能量与物质代谢；缺乏各有典型病症。

## 缺乏症速记（⚠️核对）
| 维生素 | 缺乏症 |
|---|---|
| B1 硫胺素 | **脚气病**、Wernicke 脑病 |
| B2 核黄素 | 口角炎、舌炎、阴囊炎 |
| B3 烟酸 | **糙皮病**（3D：皮炎/腹泻/痴呆） |
| B6 吡哆醇 | 皮炎、贫血、惊厥 |
| B9 叶酸 | 巨幼细胞贫血、胎儿神经管缺陷（见[[叶酸]]） |
| B12 钴胺素 | 巨幼细胞贫血 + 神经损害 |

- 辅酶对应见 [[维生素与辅酶]]

## 🎯 高频考点
- [ ] B1→脚气病、B3→糙皮病（3D）
- [ ] B9/B12→巨幼细胞贫血（B12 还伴神经症状）
- [ ] B2→口腔/阴囊炎

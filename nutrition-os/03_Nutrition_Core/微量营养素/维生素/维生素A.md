# 维生素A

## 一句话
> 脂溶性维生素，关键作用是**维持暗视觉**与上皮完整。

## 核心要点
- 形式：视黄醇（动物性）；β-胡萝卜素（植物性，维生素A原）
- 功能：视觉（视紫红质）、上皮细胞分化、免疫、生长
- **缺乏**：夜盲症、干眼症、皮肤干燥角化
- **过量**：急/慢性中毒（脂溶性会蓄积）；孕早期过量致畸 ⚠️
- 来源：动物肝、蛋黄、深色蔬果（胡萝卜素）

## 🎯 高频考点
- [ ] 缺乏 → 夜盲症（最经典）
- [ ] β-胡萝卜素是维生素A原
- [ ] 脂溶性可蓄积中毒

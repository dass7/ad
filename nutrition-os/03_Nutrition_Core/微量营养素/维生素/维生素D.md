# 维生素D

## 一句话
> 脂溶性维生素/激素原，核心作用是**促进钙磷吸收**、维持骨骼健康。

## 核心要点
- 来源：**皮肤经紫外线合成**（主要）+ 食物（鱼肝油、蛋黄）
- 活化：肝→25-(OH)D，肾→1,25-(OH)₂D（活性形式）
- 功能：促肠道钙磷吸收、骨矿化
- **缺乏**：儿童**佝偻病**、成人**骨软化症**、老年骨质疏松
- 过量：高钙血症（脂溶性可中毒）

- [ ] 促钙磷吸收、与[[钙]]协同
- [ ] 缺乏：佝偻病 / 骨软化
- [ ] 皮肤日照合成 + 肝肾活化两步

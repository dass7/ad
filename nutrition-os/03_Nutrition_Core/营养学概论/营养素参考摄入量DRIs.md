# 营养素参考摄入量DRIs

## 一句话
> DRIs = 一组评价和规划膳食的参考值，回答「吃多少够」与「吃多少算多」。

## 五个核心指标
| 缩写 | 全称 | 通俗理解 |
|-----|------|---------|
| **EAR** | 平均需要量 | 满足**50%**人群 |
| **RNI** | 推荐摄入量 | 满足**97%–98%**人群（个体目标） |
| **AI** | 适宜摄入量 | EAR 资料不足时的替代推荐 |
| **UL** | 可耐受最高摄入量 | **安全上限**，超之风险升 |
| **AMDR** | 宏量营养素可接受范围 | 三大产能营养素供能比% |

- 关系：**EAR < RNI < UL**；RNI = EAR + 2SD（变异系数10%时 RNI≈1.2×EAR）。⚠️核对

## AMDR 供能比（⚠️以最新版为准）
碳水 50%–65% ｜ 脂肪 20%–30% ｜ 蛋白质 10%–20%

## 🎯 高频考点
- [ ] 五指标缩写与定义（最爱考 EAR=50% vs RNI=97%–98%）
- [ ] RNI = EAR + 2SD
- [ ] UL 是安全上限、非推荐量
- [ ] AMDR 三大供能比

## 🧠 记忆 & 误区
- EAR 一半、RNI 几乎全、AI 顶替、UL 别越线、AMDR 看比例。
- ❌RNI 是最低量 → ✅是覆盖绝大多数人的充足目标。

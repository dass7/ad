# 营养素参考摄入量DRIs

## 📌 概念
> DRIs（Dietary Reference Intakes）= 一组用于评价和规划膳食的营养素摄入量参考值。
> 回答两个问题：「吃多少才够？」和「吃多少算多？」

## 🧱 五个核心指标
| 缩写 | 全称 | 通俗理解 | 用途 |
|-----|------|---------|------|
| **EAR** | 平均需要量 | 满足**半数（50%）**人群需要的量 | 制定 RNI 的基础、评估群体 |
| **RNI** | 推荐摄入量 | 满足**绝大多数（97%–98%）**人需要 | 个体摄入目标 |
| **AI** | 适宜摄入量 | EAR 资料不足时的替代推荐（观察得来） | 无法定 RNI 时用 |
| **UL** | 可耐受最高摄入量 | **安全上限**，超过风险增加 | 防过量 |
| **AMDR** | 宏量营养素可接受范围 | 三大产能营养素占总能量的%区间 | 膳食结构 |

> 关系记忆：**EAR < RNI ≤ ……  < UL**；RNI = EAR + 2 个标准差（SD）。⚠️核对公式表述。

## 🔬 本质
个体真实需要量无法直接测，只能用人群统计分布来「圈」一个既够用又安全的范围。
EAR 是分布的中点，RNI 把它推到覆盖几乎所有人的高位，UL 则画出安全红线。

## ⚡ AMDR（宏量营养素供能比）
> ⚠️以下为常用记忆区间，**精确值以最新版 DRIs 为准**：
- 碳水化合物：约 **50%–65%** 总能量
- 脂肪：约 **20%–30%** 总能量
- 蛋白质：约 **10%–20%** 总能量

## 🎯 高频考试点
- [ ] 五个指标的英文缩写与定义（最爱考 EAR vs RNI 的人群覆盖率）
- [ ] EAR 满足 50%、RNI 满足 97%–98%
- [ ] RNI = EAR + 2SD 的逻辑
- [ ] UL 的含义（不是推荐量，是安全上限）
- [ ] AMDR 三大营养素供能比
> #exam/highfreq

## 🧠 记忆方法
> **EAR 一半、RNI 几乎全、AI 没数据时顶上、UL 别越线、AMDR 看比例。**

## ⚠️ 常见误区
1. ❌ RNI 是「最低需要量」 ✅ RNI 已覆盖 97%–98% 人群，是充足目标而非最低。
2. ❌ 越接近 UL 越好 ✅ UL 是上限，长期接近有过量风险。

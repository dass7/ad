# DRIs — 膳食营养素参考摄入量

## 📌 概念

**DRIs（Dietary Reference Intakes，膳食营养素参考摄入量）**：
一组每日平均膳食营养素摄入量的参考值，包括四个参数：EAR、RNI、AI、UL。

用于**评估**个体/群体营养状况，以及**计划**个体/群体膳食。

---

## 🔬 本质

DRIs是营养学界对"**一个健康人每天应该吃多少这种营养素**"这个问题给出的**科学依据系统**。

它不是一个数字，而是**四个不同用途的数值框架**：

```
人群分布图：
          低←————————————→高
EAR(50%) RNI(97-98%)
   ↑          ↑
适合一半     适合几乎所有
健康人群     健康人群

AI：当EAR和RNI无法确定时使用的估算值
UL：安全上限（超过此值可能有毒副作用）
```

---

## 🫀 四个核心参数详解

| 参数 | 全称 | 定义 | 适用场景 |
|-----|------|------|---------|
| **EAR** | 估计平均需要量 Estimated Average Requirement | 满足**50%**健康个体需要的摄入量 | 评估**群体**摄入量是否充足；制定RNI的基础 |
| **RNI** | 推荐摄入量 Recommended Nutrient Intake | 满足**97-98%**健康个体需要量（=EAR+2SD） | 作为**个人**每日摄入目标 |
| **AI** | 适宜摄入量 Adequate Intake | 当EAR和RNI无法准确计算时，通过观察/实验得到的估计值 | 没有足够数据制定EAR/RNI时使用 |
| **UL** | 可耐受最高摄入量 Tolerable Upper Intake Level | 对几乎所有人不引起不良反应的最高日摄入量 | 评估摄入过多的风险上限 |

**2013年新增**：
| 参数 | 全称 | 说明 |
|-----|------|------|
| **AMDR** | 宏量营养素可接受范围 | 碳水/脂肪/蛋白质的供能比范围 |
| **EER** | 能量估计需要量 | 维持能量平衡所需的能量 |
| **PI-NCD** | 降低慢性病风险的摄入量 | 新增，针对慢性病预防 |

---

## ⚠️ 常见误区

| ❌ 误区 | ✅ 正确 |
|--------|--------|
| RNI是"最低需要量" | RNI是满足97-98%人需求的推荐量，远高于最低需要量（EAR） |
| 没达到RNI就是营养缺乏 | 没达到RNI只是"可能不足"；真正缺乏需要结合生化指标和临床症状判断 |
| UL是"推荐摄入量的上限" | UL是安全上限，不是推荐目标；实际推荐量（RNI/AI）通常远低于UL |
| AI比RNI低 | AI和RNI没有高低之分，AI是在数据不足时使用的替代值，可能高于或低于RNI |

- [ ] ★★★★★ 四个参数的英文缩写和中文名（必背）

> **经典考题陷阱**：
> "用哪个参数评估某地区人群某营养素摄入是否充足？" → **EAR**（不是RNI）
> "某营养素没有EAR，用哪个参数作为个人摄入目标？" → **AI**（不是RNI）

---

## 🏥 临床应用

**营养评估时**：
- 个人摄入量 < EAR → 摄入不足的可能性>50%，需要关注
- 个人摄入量 在EAR-RNI之间 → 可能不足，结合生化指标
- 个人摄入量 ≥ RNI → 基本充足
- 个人摄入量 > UL → 可能过多，评估风险

**膳食计划时**：
- 以RNI（或AI）为目标
- 确保不超过UL

---

## 🧠 记忆方法

```
EAR → "一半人够用的量"（50%分位数）
RNI → "几乎所有人够用的量"（97-98%分位数）
AI  → "没数据时的估算"（Alternative when Insufficient data）
UL  → "安全上限，超过可能有害"（Upper Limit）

顺序记忆：EAR < RNI < UL（三者是递增关系）
AI的位置不固定，取决于具体营养素
```

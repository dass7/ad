# 知识库使用指南

## 文件夹说明

| 文件夹 | 作用 |
|--------|------|
| `MOC/` | 地图文件，总览和分类索引 |
| `Effects/` | 50个心理学效应的详细笔记 |
| `Concepts/` | 核心心理学概念（如启发式、前景理论）|
| `Applications/` | 效应在特定领域的应用汇总 |
| `Case Studies/` | 真实案例深度分析 |
| `Templates/` | 新建效应笔记的模板 |

## 推荐学习流程

### 第一次进入知识库
1. 从 [[00-心理学效应总览]] 开始浏览全貌
2. 选择最感兴趣的类别进入对应 MOC
3. 每天精读 1-2 个效应

### 深度学习单个效应
1. 阅读「一句话定义」→ 确保理解核心概念
2. 阅读「核心机制」→ 理解为什么会这样
3. 阅读「现实案例」→ 建立直觉感知
4. 阅读「如何避免」→ 提炼可操作策略
5. 在「我的思考」部分写下个人感悟

### 复习策略
- 每周用 Dataview 仪表盘查看已读效应
- 对每个效应评分（掌握程度1-5）
- 定期回顾「与其他效应的关联」，建立效应网络

## 标签系统

- `#心理学效应` — 所有效应文件
- `#认知偏差` / `#社会心理学` / `#行为心理学` / `#决策心理学` / `#情绪心理学` / `#人际心理学` — 类别标签
- `#MOC` — 地图文件
- `#待复习` — 自己添加的复习标记

## Obsidian 功能推荐

- **图谱视图**：查看效应之间的双向链接网络
- **Dataview插件**：使用仪表盘文件进行动态查询
- **Templater插件**：使用模板快速创建新笔记
- **Spaced Repetition插件**：将效应制作成间隔复习卡片
